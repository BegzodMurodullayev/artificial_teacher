## 📋 ARXITEKTURA XARITASIDAN OLINGAN TALABLAR:

### 1️⃣ Mafia O'yini Flow (Xaritaga mos):
```
🎮 Guruh O'yinlari → 🕵️ Mafia → 
├─ Rollar taqsimlash (private message)
├─ Tun: Mafia ovoz (inline keyboard)
├─ Kun: Muzokara + Sinonim/Antonim mini-o'yin
├─ Bot avtomatik phase o'tkazadi
├─ G'olib aniqlash → Ball mukofot
└─ Leaderboard yangilanadi
```

### 2️⃣ Sinonim/Antonim Integratsiyasi (🔤 So'z Topish):
```
🔤 So'z Topish → Antonim/Sinonim
├─ Bot so'z yuboradi (levelga mos: A1-C2)
├─ User chatga javob yozadi yoki inline button bosadi
├─ Bot tekshiradi: synonyms_antonyms.json yoki AI API
├─ To'g'ri → +ball, Noto'g'ri → -ball
├─ Natija → 🏆 Mukofotlar → Bonus Ball
└─ HTML result export (mavjud pattern)
```

### 3️⃣ WebApp Integratsiyasi (🌐 Web App bo'limiga mos):
```
🌐 Web App → Sidebar → Dashboard/Focus/Tracker/Progress
├─ Yangi sahifa: /mafia (mafia.html)
├─ Tracker: "Mafia games played", "Win rate", "Synonym mastery"
├─ Progress: Canvas chartga Mafia stats qo'shish
├─ Bot Sync: sendData() orqali Mafia ball → DB
├─ Settings: Mafia notification toggle
└─ Mavjud web_app_data_handler ga yangi case: 'mafia_sync'
```

### 4️⃣ Admin Panel Kengaytmasi (🛡 Admin Panel):
```
🛡 Admin Panel → Yangi bo'lim: 🎮 Mafia Boshqaruv
├─ 📊 Dashboard: Faol o'yinlar, stats
├─ 💳 To'lovlar yoniga: 🎁 Mafia Ball sovg'alari
├─ 👥 Userlar → Mafia stats filter
├─ 📦 Rejalar → Mafia Premium access
├─ 📣 Reklama → Mafia tournament broadcast
└─ Owner Only → Mafia config (timer, roles, balls)
```

### 5️⃣ Callback Structure (🔧 Callbacks bo'limiga mos):
Mavjud pattern: `do_quiz`, `do_iq`, `do_daily`...
Yangi callback'lar:
```python
do_mafia_start      # /mafia command
do_mafia_join       # Lobbyga qo'shilish
do_mafia_role       # Rolni ko'rish (private)
do_mafia_night_kill # Mafia qurbon tanlash
do_mafia_night_check# Detective tekshiruv
do_mafia_day_vote   # Ovoz berish
do_mafia_synonym    # Sinonim javob tekshiruv
do_mafia_admin      # Admin boshqaruv
do_mafia_webapp     # WebApp ochish
do_mafia_sync       # WebApp → Bot data sync
```

## 🔧 TEXNOLOGIK CHEKLOVLAR (MUHIM):

1. **Stackni o'zgartirma**: 
   - `python-telegram-bot==21.9` (aiogram EMAS)
   - `sqlite3` + mavjud `engbot.db`
   - `httpx`, `python-dotenv`, `Jinja2`

2. **Code style'ga rioya qil**:
   - `bot/bot.py` dagi `ApplicationBuilder`, `ContextTypes`, async handler pattern
   - `handlers/quiz.py` dagi callback structure'ni takrorla
   - `utils/` dagi logging, error handling, HTML export funksiyalaridan foydalan

3. **Database migration (xavfsiz)**:
   ```sql
   -- Yangi table'lar, mavjudlarni o'zgartirmasdan:
   CREATE TABLE IF NOT EXISTS mafia_games (...);
   CREATE TABLE IF NOT EXISTS mafia_players (...);
   CREATE TABLE IF NOT EXISTS synonym_answers (...);
   -- Migration script: bot/database/migrations/001_mafia.py
   ```

4. **WebApp xavfsizligi**:
   - Mavjud `webapp/utils.js` dagi `initData` validate funksiyasidan foydalan
   - Yangi `mafia.js` da: `Telegram.WebApp.sendData('mafia_sync', data)`
   - Botda: `web_app_data_handler` ga `case 'mafia_sync':` qo'sh

5. **Inline keyboards**:
   - Mavjud `keyboards.py` pattern'ini ishlat
   - Har bir callback_data prefix: `mafia:` (masalan: `mafia:night_kill:12345`)
