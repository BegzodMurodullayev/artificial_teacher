## 1. 🕵️ MAFIA O'YINI

### O'yin Tuzilishi

**O'yinchilar soni:** 6-12 kishi
**Rollar:**
- **Mafia** (2-3 ta) - bir-birini biladi
- **Komissar** (1 ta) - tinch aholi tarafida
- **Shifokor** (1 ta) - tinch aholi tarafida
- **Tinch aholi** (qolganlari) - rollarni bilmaydi

### Tekshirish Logikasi

```python
def check_answer(user_answer, correct_error, correct_word):
    """
    Foydalanuvchi javobini tekshirish
    
    Qabul qilinadigan formatlar:
    - "go → went"
    - "go -> went"
    - "go - went"
    - "go/went"
    - "went" (faqat to'g'ri variant)
    """
    
    user_answer = user_answer.lower().strip()
    correct_answer = f"{correct_error} → {correct_word}"
    
    # To'liq javob
    if user_answer == correct_answer.lower():
        return True, "To'liq to'g'ri!"
    
    # Faqat to'g'ri so'z
    if user_answer == correct_word.lower():
        return True, "To'g'ri, lekin xato so'zni ham ko'rsating"
    
    # Xato so'z + to'g'ri so'z (boshqa ajratuvchi)
    import re
    pattern = rf"{re.escape(correct_error)}\s*[-→/]\s*{re.escape(correct_word)}"
    if re.match(pattern, user_answer, re.IGNORECASE):
        return True, "To'g'ri!"
    
    return False, f"Noto'g'ri. To'g'ri javob: {correct_answer}"
```

```python
def check_translation(user_translation, correct_translation, ai_model):
    """
    Tarjimalarni solishtirish
    
    1. To'g'ri javob bilan solishtirish
    2. AI orqali ma'noni tekshirish
    3. Grammatikani tekshirish
    4. Ball berish
    """
    
    # 1. Exact match
    if user_translation.lower().strip() == correct_translation.lower().strip():
        return 'perfect', 20, "A'lo tarjima!"
    
    # 2. AI semantic check
    ai_result = ai_model.check_semantic_similarity(
        user_translation,
        correct_translation
    )
    
    # 3. Grammar check
    grammar_errors = grammar_checker.check(user_translation)
    
    # 4. Scoring
    if ai_result.similarity > 0.9 and len(grammar_errors) == 0:
        return 'perfect', 20, "A'lo!"
    elif ai_result.similarity > 0.8 and len(grammar_errors) <= 2:
        return 'good', 15, "Yaxshi, lekin kichik xatolar bor"
    elif ai_result.similarity > 0.6:
        return 'partial', 5, "Qisman to'g'ri"
    else:
        return 'wrong', 0, "Noto'g'ri tarjima"
```
